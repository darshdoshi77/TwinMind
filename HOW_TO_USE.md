# How to Use TwinMind

## Overview
TwinMind is your AI-powered "Second Brain" that helps you remember and query everything you've saved - documents, audio, video, web articles, and notes.

## Uploading Content

### Supported Formats
- **Audio/Video**: MP3, M4A, WAV, MP4, WEBM
- **Documents**: PDF, DOCX, TXT, MD (Markdown)
- **Web**: Any URL (articles, blog posts, etc.)
- **Text**: Plain text notes

### Upload Process
1. Click **"Add Content"** button
2. Select your file type tab (File/Web/Text)
3. Upload or paste your content
4. You'll get a confirmation message

## Asking Questions

Once your content shows "Ready" status, you can ask questions in the chat!

### Example Questions

#### About Uploaded Content
- "What videos did I upload?"
- "Summarize the audio I added yesterday"
- "What did the video say about machine learning?"
- "Tell me the main points from the document I saved"

#### Content Search
- "What did I save about [topic]?"
- "Find information about [subject] in my uploads"
- "What articles did I read about [keyword]?"

#### Summaries & Analysis
- "Give me a summary of everything I've uploaded this week"
- "What are the key themes in my saved content?"
- "Compare the information from different sources I've added"

#### Specific Queries
- "What were the important dates mentioned in my content?"
- "List all the projects mentioned in my uploads"
- "What recommendations were made in the articles I saved?"

## Tips

1. **Wait for Processing**: Don't ask questions about content that's still "Processing"
2. **Be Specific**: More specific questions get better answers
3. **Use Keywords**: Mention topics, subjects, or keywords from your content
4. **Check Sources**: Use the "Sources" button to see what's been indexed

## Troubleshooting

### Content Not Showing Up in Answers?
- Check the "Sources" button - make sure status is "Ready"
- Wait a few more minutes for processing to complete
- Try asking more general questions first

### Upload Failed?
- Check file size (max 500MB)
- Ensure file format is supported
- Check browser console for error messages
- Verify backend is running

### Processing Taking Too Long?
- Large video files (5+ minutes) can take 5-10 minutes
- Audio files typically process in 1-3 minutes
- Documents and text are usually instant
- Check backend logs if stuck in "Processing" for >15 minutes
